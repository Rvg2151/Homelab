Change Detection LXC
Change Detection is a service that allows you to monitor changes to web pages and receive notifications when changes occur. It can be used for a variety of purposes such as keeping track of online price changes, monitoring news websites for updates, or tracking changes to online forums.

To create a new Proxmox VE Change Detection LXC, run the command below in the Proxmox VE Shell.
To Update Change Detection, run the command below (or type update) in the LXC Console.

Copy

bash -c "$(wget -qLO - https://github.com/tteck/Proxmox/raw/main/ct/changedetection.sh)"

⚡ Default Settings: 512MiB RAM - 2GB Storage - 1vCPU ⚡
Change Detection Interface: IP:5000
